# Comandos SQL que todo profissional de dados precisa dominar

As Querys que mais são utilizadas no dia a dia de um profissional de dados.

Hoje compartilho o guia que eu gostaria de ter tido no começo:

✅ Carregar todos os dados? → SELECT *

✅ Filtrar linhas? → WHERE

✅ Escolher só as colunas que importam? → SELECT col1, col2

✅ Ordenar resultados? → ORDER BY

✅ Agrupar e agregar? → GROUP BY + funções de agregação

✅ Contar registros? → COUNT(*)

✅ Calcular média? → AVG()

✅ Somar valores? → SUM()

✅ Remover duplicatas? → DISTINCT

✅ Combinar tabelas? → JOIN

✅ Criar coluna calculada? → SELECT col * 0.1 AS nova_col

✅ Renomear coluna? → AS

Com elas você resolve 80% dos problemas do dia a dia.
